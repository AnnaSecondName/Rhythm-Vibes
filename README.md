# Ритм-игра со стрелочками на HTML5

*Для локального запуска: откройте файл `index.html` в браузере*

## Об игре
Rhythm Arrow Game - это увлекательная браузерная ритм-игра, созданная по мотивам чистых HTML5, CSS3 и JavaScript. Задача игрока заключается в совпадении нажатия стрелок на клавиатуре или экране в момент, когда стрелки падают на целевую линию. Особенность игры заключается в том, что по мере роста набранных очков скорость движения падающих стрелков увеличивается, а их частота повышается, что удерживает внимание игрока и подстегивает его на вооруженные действия. Разработка продемонстрировала современные возможности веб-технологий для создания интерактивных игровых приложений без стронних библиотек и фреймворков, предлагая плавные движения, многопользовательские комбо-системы и визуальную комбинаторику для оценки точности нажатий.

### Игровой процесс
Управление происходит через стрелочки на клаиатуре, либо через кнопки-стрелочки на станице сайта. С увеличением количества очков скорость прокрутки карты стрелок будет увеличиваться в арефметической прогрессии. Последовательные попадания увелиичивают счетчик множителя, который увеличивает очки в соответсвии с набранным комбо. При промоще счетчик сбрасывается до значений по умолчанию.
